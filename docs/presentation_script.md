## 라이브 데모 (2:30~5:00)

### 데모 1: 대학가 원룸 (페르소나 A — 단국대 새내기)
**조작 순서:**
1. 안전지도 탭 → 안서동 마커 클릭 → "종합 57.1점, 금융 위험"
2. 시뮬레이터 탭 → 안서동 / 5,000만원 / 33㎡ / 2000년식
3. 결과: 🔴 **위험 85.8%** → SHAP 바 차트 "건물연령이 가장 큰 위험 요인"
4. "이 학생에게 안전한 대체 매물은?"

> 멘트: "26년 된 원룸에 5천만 원 보증금. AI가 85.8% 위험이라고 경고합니다.
> 바로 아래 SHAP 그래프를 보시면 — 건물 노후가 가장 큰 위험 요인입니다."

### 데모 2: 신도시 아파트 (페르소나 B — 사회초년생)
**조작 순서:**
1. 시뮬레이터 → 불당동 / 1.5억 / 84㎡ / 2018년식
2. 결과: 🟡 **주의 30.1%** → "건물 신축이 안전 요인, 하지만 거래연도가 위험 방향"
3. 대시보드 탭 → 불당동 레이더 차트 → "8축 중 치안·편의 양호, 금융 주의"

> 멘트: "같은 1.5억이라도 불당동 신축은 30%, 원성동 구축은 40%.
> 어디에 계약하느냐에 따라 위험이 3배 차이납니다."

### 데모 3: 행정 관점 (페르소나 C — 천안시 담당자)
**조작 순서:**
1. 추세분석 탭 → 위험 경보 19개 동 표시
2. "입장면 기로리 115%, 6개월 +40.5%" → "이 동네 선제 점검 필요"
3. 이상탐지 탭 → 동별 이상 거래 비율 → "불당동 16%가 이상 거래"
4. "성거읍 요방리 육영원룸 — 전세가율 435%의 극단적 이상"

> 멘트: "행정에서는 '어디부터 점검할 것인가'가 핵심입니다.
