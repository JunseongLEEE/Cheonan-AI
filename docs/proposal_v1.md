# 천안 청년 자취방 안전지도: AI 기반 종합 주거안전 진단 서비스

## 1. 문제정의

### 1.1 천안 청년 주거 위기의 실태

천안시는 청년인구(18~39세) **197,572명**(전체 661,615명의 29.9%)으로, 전국 평균(26.1%)·충남(23.7%)을 크게 상회하는 청년 집중 도시다. 그러나 청년 주택소유율은 **13.9%**에 불과하며, **약 86%가 무주택 임차 상태**로 전세·월세 시장에 의존하고 있다.

이러한 취약한 주거 기반 위에 전세사기가 확산되어, 2022~2024년 **288세대, 145억원**의 피해가 발생했다. 주로 다가구·오피스텔·도시형생활주택 등 청년 밀집 주거에서 무자본 갭투자, 업계약서, 허위 임대차계약서를 이용한 사기가 반복되고 있다.

특히 30대 청년의 **주거이전 사유 전출**이 심각하다(35~39세의 43.2%, 30~34세의 35.6%가 '주택' 사유). 주거 불만족이 인구 유출의 직접 원인이 되고 있어, 주거안전 확보는 곧 **인구 유지 정책**이다.

### 1.2 기존 서비스의 한계

| 서비스 | 한계 |
|--------|------|
| HUG 안심전세앱 | 사용자가 입력한 **1채만 진단**, 도시 전체 선제 스캔 불가 |
| 직방/다방 | 매물 중개 부가서비스, 중립적 위험평가 아님 |
| 부동산플래닛 | 상업 데이터 기반, 청년 맞춤 아님 |
| 청년몽땅정보통 | 정책 안내 중심, AI 위험예측 없음 |

**우리의 차별점**: 도시 전체를 H3 헥사곤 격자로 선제 스캔 + 종합 안전성(전세사기 + 침수 + 치안 + 교통 + 편의시설) 통합 진단

## 2. 프로젝트 비전

> "천안의 모든 자취방, 계약 전 30초 안전진단"

청년이 자취방 계약 전 깡통전세·건물 노후·침수·치안·교통·편의시설 등 **8축 종합 안전성**을 한눈에 진단받을 수 있는 AI 기반 지도 서비스를 구축한다. 동시에 천안시 행정에 위험 핫스팟 대시보드를 제공하여 선제적 점검·예산 배분의 근거 데이터로 활용한다.

### 과제 매핑
- **지역균형발전**: 구도심(동남구) vs 신도심(서북구) 청년 주거안전 격차 해소
- **국토/교통**: 대중교통 접근성 기반 안전 매물 추천

## 3. 페르소나 시나리오

### 페르소나 A: 단국대 새내기 (19세, 안서동)
> "첫 자취인데 이 보증금이 안전한지 어떻게 아나요?"

1. 안전지도에서 안서동 지역 조회 → H3 격자별 신호등(빨/노/초) 확인
2. 관심 매물 조건 입력(보증금 5,000만원, 다가구, 2000년식)
3. AI가 전세가율 85%, 건물연령 24년으로 **"위험"** 판정 + SHAP 설명
4. "더 안전한 대체 매물" 추천으로 같은 통학권 내 안전 매물 3건 제시
5. 정책 챗봇이 "청년월세 한시 특별지원" 신청 안내

### 페르소나 B: 사회초년생 (28세, 두정동)
> "전세보증금 7,000만원인데 깡통 아닌지 확인하고 싶어요"

1. 시뮬레이터에서 조건 입력 → 공시가 대비 HUG 126% 룰 자동 계산
2. 해당 건물의 동일 임대인 다물건 패턴 확인 → 그래프 탐지 경고
3. 종합 안전점수 72/100 (금융 위험 + 건물 노후 감점, 교통·편의 양호)
4. "이 건물 비슷한 조건에서 과거 전세가율 추이" 시계열 제공

### 페르소나 C: 천안시 주거정책 담당자 (B2G)
> "어느 동네를 우선 점검해야 하나요?"

1. 행정 대시보드에서 위험 핫스팟 히트맵 조회
2. 동남구 원성동·신부동 깡통전세 위험 밀집 확인
3. 노후 다가구 밀집지 + 청년 인구 밀도 교차분석
4. 안심계약 도움서비스 우선 배치 지역 근거자료로 활용

### 4.2 ETL 흐름

```
공공 API 수집 (collector/)
    ↓
XML/JSON → Pandas DataFrame 변환
    ↓
정제 (결측, 이상치, 주소 정규화)
    ↓
```

## 5. AI 모델 아키텍처

### 5.1 핵심 모델 — 실증 결과 포함

#### Model 1: 깡통전세 분류기 (LightGBM + SHAP)
- **목적**: 전세 계약의 깡통 위험도 예측
- **라벨 전략**: 전세가율 90%+ → 위험(19,310건), 60%- → 안전(14,328건), 60~90% 제외 (PU러닝 간소화)
- **피처 27개**: 보증금, 전용면적, 건물연령, 동별 역사적 위험도, 동별 건축물 통계(노후비율/내진비율/구조분포), 구별 평균 대비 비율 등
- **매매가 미사용**: 실서비스에서 매매가를 모를 수 있으므로 proxy 피처로 대체

| 실험 | AUC | F1 | 특이사항 |
|------|-----|-----|---------|
| exp_001 (baseline) | 0.9984 | 0.9881 | 매매가비율 포함 (data leakage) |
| exp_002 (매매가 제거) | 0.9929 | 0.9640 | proxy 피처 대체, 실서비스 시나리오 |
| **exp_003 (채택)** | **0.9931** | **0.9646** | 건축물대장 14개 피처 추가, 설명력 향상 |

- **SHAP Top 3 피처**: 동별 역사적 위험도(2.03), 건물연령(1.34), ㎡당 보증금(1.21)
- **설명 가능성**: SHAP force plot으로 "왜 위험한지" 피처별 기여도 시각화

#### Model 2: 8축 종합 안전점수 (5축 데이터 반영)
| 축 | 지표 | 가중치 | 데이터 반영 |
|----|------|--------|-------------|
| 금융안전 | 전세가율, 보증금/매매가 | 0.25 | 실거래 27,348건 |
| 건물노후 | 건물연령, 구조, 내진설계 | 0.15 | 건축물대장 21,669건 |
| 침수위험 | 침수흔적 거리, 지하층 | 0.10 | 미구현 (WMS only) |
| 치안 | 인구밀도, 인구당 사업체수 | 0.15 | **SGIS 31개 행정동** |
| 소방 | 소방서 접근시간 | 0.05 | 미구현 |
| 교통 | 버스정류장·역 도보거리 | 0.10 | 미구현 |
| 편의시설 | 사업체·종사자 밀도 | 0.10 | **SGIS 인구통계** |
| 환경 | 가구대비 주택비율 | 0.10 | **SGIS 주택통계** |

- 65개 동 분석 결과: 🔴빨강 1개 / 🟡노랑 47개 / 🟢초록 17개
- 가장 위험: **원성동** (44.9점) — 전세가율 94%, 건물연령 42년
- 가장 안전: **부대동·성성동** (75점대) — 신도시, 신축, 양호 인프라
- 사용자 가중치 슬라이더로 개인화 가능 (예: "치안 중시")

#### Model 3: 이상탐지 (Isolation Forest / HDBSCAN)
- 동일 임대인 다물건 + 고전세가 패턴 탐지
- 임대인-건물 이분그래프 → "빌라왕형" 연쇄 위험 식별

#### Model 4: 시계열 추세 분석 + 위험 경보
- 63개 동 월간 전세가율 추이 분석 (6,210건 월별 데이터)
- 6개월 이동평균 기반 추세 판정: 급상승/상승/정체/하락/급하락
- **실증 결과**: 전세가율 80%+ 이면서 상승 추세인 **위험 경보 19개 동** 탐지
  - 입장면 기로리 (115%, 6M +40.5%) — 가장 급격한 상승
  - 직산읍 상덕리 (125%, 6M +8.8%) — 이미 깡통 수준
  - 봉명동 (104%, 6M +9.4%) — 도심 깡통 진입
- → 시 행정이 선제적 점검 자원을 배분할 수 있는 **조기 경보 시스템**

### 5.2 시뮬레이터 (실증 데모)

### 5.3 보조 모델
- **추천 시스템**: 사용자 조건 기반 안전 매물 추천 (Content-based + k-NN)

- **등기부 OCR**: 사용자 업로드 등기부 → 위험조항 자동 탐지

## 6. 시각화 / 서비스 설계

### 6.1 B2C: 청년 사용자

1. **신호등 지도**: H3 헥사곤 격자로 동네 전체를 빨강/노랑/초록 표시 + 개별 매물 핀
2. **시뮬레이터**: "보증금 ○○만원, ○년식 다가구" 입력 → 실시간 위험도 + SHAP 설명
3. **매물 상세**: 전세가율 그래프, 건물 이력, 주변 인프라, 대체 매물 추천
4. **정책 챗봇**: "월세 지원 받을 수 있나요?" 등 자연어 상담

### 6.2 B2G: 천안시 행정

1. **위험 핫스팟 히트맵**: 동네별 깡통전세 위험 밀집 지역
2. **사기 추세 시계열**: 연도별·동별 전세가율 변화, 사기 발생 추이
3. **정책 효과 모니터링**: 행복주택·월세지원 수혜지 매핑, 안심계약 서비스 활용 현황

### 기술 스택
- **Frontend**: Next.js + Mapbox GL JS / deck.gl (H3 헥사곤) + shadcn/ui
- **Backend**: FastAPI + Python
- **시각화**: Recharts, SHAP plots
- **모바일 우선**: 청년 사용자 대상 반응형 설계
